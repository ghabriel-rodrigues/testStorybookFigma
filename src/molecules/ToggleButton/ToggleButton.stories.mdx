# ToggleButton
`ToggleButton` é um **componente composto de outros componentes (molecule)**.

`import { ToggleButton } from '@gupy/design-system-v2'`

## Acessibilidade
Com o foco no componente, é possível obter os seguintes comportamentos
  - Ao pressionar `SPACE` e/ou `ENTER`, irá realizar a ativação/desativação do botão

Para demais extensões dos comportamentos, [verifique a documentação](https://www.radix-ui.com/docs/primitives/components/toggle)
