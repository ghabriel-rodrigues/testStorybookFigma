# Toast

`Toast` é um **componente composto de outros componentes (molecule)**.

`import { Toast } from '@gupy/design-system-v2'`

## Acessibilidade

Todo `Toast` deve receber uma propriedade `label` para ser atribuído ao `aria-label` do botão de fechar.

```javascript
<Toast label="remover filtro">Vagas remotas</Toast>
```
