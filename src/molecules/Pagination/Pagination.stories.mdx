# Pagination

`Paginaton` é um **componente composto de outros componentes (molecule)**.

`import { Paginaton } from '@gupy/design-system-v2'`

## Acessibilidade

Utilize o atributo `ariaLabelPagination` para indicar o que se refere a Paginação. Ex: Paginação do resultado da busca.

Utilize o atributo `ariaLabelPageButton` para indicar o botão que recebe o valor da página. Obs: O valor da página é preenchido automaticamente. Ex: Página 3.

Utilize o atributo `ariaLabelNextButton` para indicar o botão de próximo. Ex: Vá para próxima página.

Utilize o atributo `ariaLabelPrevButton` para indicar o botão de anterior. Ex: Vá para página anterior.
