# LabeledTextArea

`LabeledTextArea` é um **componente composto de outros componentes (molecule)**.

`import { LabeledTextArea } from '@gupy/design-system-v2'`

## Acessibilidade

- Informe o valor da propriedade `id`, que será repassado ao componente Label em seu atributo `htmlFor`,
  fazendo assim a ligação da label com o textarea.

- Também é possível utilizar o próprio attr `label` , que será o texto exibido pela label.

- Caso seja necessário utilizar a propriedade `helperMessage`, o próprio componente já possui a inteligência
  de setar os parâmetros corretos da wai-aria para indicar que aquele texto esta relacionado ao uso do textarea
