# InputFile

`InputFile` é um **componente base (atom)**.

`import { InputFile } from '@gupy/design-system-v2'`

## Acessibilidade

Utilize o componente informando pelo menos a property onFileUpload e, de preferência, o id.

Esse componente utiliza apenas componentes HTML, então já contempla os funcionamentos acessíveis providos pelo HTML.

Exemplo:

`<InputFile onFileUpload={handleFileInput} id="file-upload" />`
