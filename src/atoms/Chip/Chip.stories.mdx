# Chip

`Chip` é um **componente base (atom)**.

`import { Chip } from '@gupy/design-system-v2'`

## Acessibilidade

O componente `Chip` pode receber uma propriedade `closeLabel` para ser atribuído ao `aria-label` do botão de fechar.

```javascript
<Chip label="remover filtro">Vagas remotas</Chip>
```
