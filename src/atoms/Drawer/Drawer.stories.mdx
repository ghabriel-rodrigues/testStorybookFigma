# Drawer

`Drawer` é um **componente base (atom)**.

`import { Drawer, Target, Trigger } from '@gupy/design-system-v2'`

## Acessibilidade

Drawer é um componente controlado por um [disclosure](https://www.w3.org/TR/wai-aria-practices-1.1/#disclosure).
Para ele funcionar via teclado e via leitores de tela, precisamos passar um _id_ único para o componente _Drawer_, e utilizar
os componentes _Trigger_ e _Target_ para compor suas funcionalides.
Dessa forma, todas as regras necessarias para um disclosure seram aplicadas corretamente.
