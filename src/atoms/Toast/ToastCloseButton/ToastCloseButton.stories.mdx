# ToastCloseButton

`ToastCloseButton` é um **componente base (atom)**.

`import { ToastCloseButton } from '@gupy/design-system-v2'`

## Acessibilidade

Todo `ToastCloseButton` deve receber uma propriedade `label` para ser atribuído ao `aria-label` do botão de fechar.

```javascript
<ToastCloseButton label="remover filtro">Vagas remotas</ToastCloseButton>
```
