# ToastActionButton

`ToastActionButton` é um **componente base (atom)**.

`import { ToastActionButton } from '@gupy/design-system-v2'`

## Acessibilidade

Todo `ToastActionButton` deve receber uma propriedade `label` para ser atribuído ao `aria-label` do botão de fechar.

```javascript
<ToastActionButton label="remover filtro">Vagas remotas</ToastActionButton>
```
