# Tip

`Tip` é um **componente base (atom)**.

`import { Tip } from '@gupy/design-system-v2'`

## Acessibilidade

- O componente Tip exibe um mensagem de texto com string simples ou composta por elementos HTML. Para garantir a acessibilidade do componente, utilize apenas outros elementos acessíveis em sua composição.

- Os ícones implementam a propriedade `aria-hidden` para não haver problemas com leitores de tela.

Exemplo de componente acessível:
```
<Tip type="info">
  <Paragraph><strong>Sucesso:</strong> Mensagem informativa.</Paragraph>
</Tip>
```
