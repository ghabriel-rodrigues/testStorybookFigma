# Input

`Input` é um **componente base (atom)**.

`import { Input } from '@gupy/design-system-v2'`

## Acessibilidade

Utilize placeholders exemplificando o tipo de dado que deve ser inserido

Exemplo:

`<Input placeholder="Agnaldo Silva" id="username" name="username" />`
