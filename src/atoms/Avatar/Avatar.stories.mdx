# Avatar

`Avatar` é um **componente base (atom)**.

`import { Avatar } from '@gupy/design-system-v2'`

## Acessibilidade

Caso você queira que o leitor de tela leia seu avatar, use a propriedade `label` com uma descrição acessível.

```javascript
<Avatar label="Homem de camisa branca, usando barba e óculos sorrindo ao nascer do sol, em frente às pirâmides egípcias" />
```
