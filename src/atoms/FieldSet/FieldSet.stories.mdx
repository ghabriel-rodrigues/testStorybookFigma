# FieldSet

`FieldSet` é um **componente base (atom)**.

`import { FieldSet, Legend } from '@gupy/design-system-v2'`

## Acessibilidade

O componente `FieldSet` fornece um contêiner para controles de formulário relacionados, e o componente `Legend` atua como um cabeçalho para identificar o grupo.

## Propriedades

Contém todas as propriedades da tag HTML `fieldset`.
