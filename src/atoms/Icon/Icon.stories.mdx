# Icon

`Icon` é um **componente base (atom)**.

`import { Icon } from '@gupy/design-system-v2'`

## Acessibilidade

Todos os ícones implementam por padrão a propriedade `aria-hidden`, para que não haja problemas com os leitores de tela.
