# Label

`Label` é um **componente base (atom)**.

`import { Label } from '@gupy/design-system-v2'`

## Acessibilidade

Utilize o atributo `htmlFor` para indicar o sentido semantico de inputs,
auxiliando utilizadores de screen-readers entenderem precisamente seu uso, caso necessário.

Para tal, também é possível utilizar o próprio attr `ariaLabel` do `<Input/>`, verifique o uso se necessário.
