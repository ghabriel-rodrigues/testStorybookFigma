# Dialog

`Dialog` é um **componente composto de outros componentes (organism)**. Existem 5 variações básicas de tamanho: `xs` (extra small),
`sm` (small), `md` (medium) e `lg` (large) e `fullScreen`.

`import { Dialog } from '@gupy/design-system-v2'`

**Este componente é interdependente dos componentes listados abaixo**

- `Radix UI Dialog.Primitive`
- `@atoms/Icon`
- `@atoms/Button`

## Acessibilidade

Utilizando a implementação da [RADIX UI - Dialog](https://www.radix-ui.com/docs/primitives/components/dialog), esse componente suporta todos os comportamentos assistivos requeridos pela A11Y, inferindo referências tal qual requerido na [documentação a11y](https://www.w3.org/WAI/ARIA/apg/patterns/dialogmodal/). Favor consultar a documentação da Radix para mais informações sobre os atributos e métodos desse componente.
