# CardCollapse

`CardCollapse` é um **componente composto de outros componentes (organism)**..

`import CardCollapse from '@gupy/design-system-v2'`

**Este componente é interdependente dos componentes listados abaixo**

* `Radix UI Collapsible.Primitive`
* `@atoms/Icon`
* `@atoms/Button`
* `@atoms/Heading`
* `@atoms/Divisor`

## Acessibilidade

O componente `CardCollapse` utiliza a implementação da [RADIX UI - Collapsible](https://www.radix-ui.com/docs/primitives/components/collapsible).
Esse componente suporta os comportamentos assistivos requeridos pela A11Y seguindo o padrão [Disclosure](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/).
Consulte a documentação da Radix para mais informações sobre os atributos e métodos desse componente.

Quando o conteúdo está visível, o elemento botão possui o atributo aria-expanded como `true`. Quando o conteúdo está escondido, o atributo é `false`.
Também é possível escolher um texto de leitura para o botão de expansão utilizando a prop `expandButtonAriaLabel`, que define o atributo `aria-label` do mesmo.
